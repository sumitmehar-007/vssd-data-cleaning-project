# VSSD Data Cleaning Project # 

## 📌 Project Overview
This project demonstrates a complete data cleaning process using Python and Pandas. We check in dataset missing values, duplicate records, inconsistent text, incorrect data types, and formatting issues. After cleaning, the dataset became ready for analysis and visualization.

---

## 🎯 Objectives
- Clean raw dataset
- Handle missing values
- Remove duplicate records
- Standardize text values
- Correct inconsistent data
- Convert data into proper data types
- Export the cleaned dataset

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/vssd-data-cleaning-project.git
```

2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run all notebook cells.
